## Модули

### api/chatRoutes.ts
Маршруты для работы с чатами:
- `GET /` - получение всех чатов
- `POST /chats` - создание чата
- `POST /chat/:id/send` - отправка сообщения в чат
- `GET /chats/:id` - получение истории чата
- `DELETE /chats/:id` - удаление чата
