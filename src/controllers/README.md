# Controllers Module

## Правила документации
- Только необходимые зависимости и их назначение
- Только ключевые методы и их назначение
- Без примеров кода
- Без детального описания реализации
- Без избыточной информации

## Зависимости
- `chatService` - работа с чатами
- `messageService` - обработка сообщений

## Модули

### chatController.ts
Обработка HTTP запросов для чатов:
- `getAllChats` - получение всех чатов
- `createChat` - создание чата
- `getChatHistory` - получение истории чата
- `deleteChat` - удаление чата
- `sendToChat` - отправка сообщения в чат
