# Services Module

## Правила документации
- Только необходимые зависимости и их назначение
- Только ключевые функции и их назначение
- Без примеров кода
- Без детального описания реализации
- Без избыточной информации

## Зависимости
- `mongoose` - работа с моделью Chats
- `openai` - клиент для OpenAI API (используется в openRouterService)

## Модули

### chatService.ts
CRUD операции для чатов: получение всех, создание, получение истории, удаление, добавление сообщений.

### messageService.ts
Обработка отправки сообщений: получение истории чата, формирование запроса к AI, обновление истории, возврат ответа.

### openRouterService.ts
Интеграция с OpenRouter API для получения ответов от AI модели.
